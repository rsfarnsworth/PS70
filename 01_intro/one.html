  • Primary Project Idea

  • For this assignment, I followed the in class suggestion of searching through previous student's webpages for interesting template designs. Many were really cool, but the style and font of Bella Tarantino's website was most appealing to me so I've chosen to use that as the main template. I'm planning to update this as I go along and get better with HTML/CSS. Additionally, as Bella notes, it'd be fun to make this website design more functional accross devices as I get better with the code here.

    Thus far, I've changed the directory organization to emphasize the weekly structure of the course, but this isn't a visual, web user, change. I've also made some changes to page layout and function in week 1 so I can learn more of how this works.

  • One of my biggest hobbies is playing and listening to music. Recently my musical focus has re-centered around playing guitar, which has sparked the two ideas for final projects I currently have. The seemingly most accessible of which (though I'm still worried about cost) would be the creation of an effects pedal, or multi-pedal. I haven't fully decided on the number of effects, or which effects, but I was thinking I could include two or three, LEDs or a small LED screen to show which effect/combination of effects are on. I'd need switches to step on, potentiometers or volume/levels, and a microcontroller could be used to control which effect circuitry is powered and sending an output although the circuitry is probably more complicated than I realize at the moment. One effect I'd liked to have is a wha pedal with the variable foot lever. An EQ pedal might be within range.

    The image below is of a very popular multi-effects pedal, my friend actually has this one, but this gives the idea of the wah gas pedal type thing I'd want on the piece as well as the kinds of switches I'd want to include, though I know mine won't be nearly as sophisticated.
